# 📰 AI Fake News Detection

An AI-powered Fake News Detection web application built using Machine Learning and Flask.

## 🚀 Features

- Detects Fake and Real News
- Machine Learning Model (98.57% Accuracy)
- Confidence Score
- Prediction History
- PDF Report Download
- Clean and Responsive UI

## 🛠️ Tech Stack

- Python
- Flask
- Scikit-learn
- Pandas
- NumPy
- HTML
- CSS

## 📂 Project Structure

Fake-News-Detection/
│── app.py
│── train_model.py
│── requirements.txt
│── README.md
│── dataset/
│── model/
│── static/
│── templates/

## ▶️ Installation

```bash
git clone https://github.com/aamirsalam97-design/Fake-News-Detection.git
cd Fake-News-Detection

pip install -r requirements.txt

python app.py
```
